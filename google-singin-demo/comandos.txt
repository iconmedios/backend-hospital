https://github.com/johnpapa/lite-server

1. npm install -g lite-server

2. Dentro de package.json...
  "scripts": {
    "dev": "lite-server"
  },

npm run dev

3. Para cambiar puerto

crear archivo config.json

{
  "port": 4200,
  "files": ["./src/**/*.{html,htm,css,js}"],
  "server": { "baseDir": "./src" }
}


4. en package.json escribir la ruta del archivo config
